System Architecture

Bill of Materials

ESP32-C6
RISC-V 32-bit | WiFi 6

Central processing unit handling inverse kinematics and mesh networking protocol.

PCA9685
16-Channel PWM | I2C

Dedicated servo driver. Offloads PWM generation to ensure jitter-free leg movement.

SPIDER CHASSIS
PETG / ABS Composite

Main structural frame. Hexapod configuration with 3DOF per leg (18DOF total).

MG90S SERVOS
Metal Gear | High Torque

Micro servos for joint actuation. Requires 12-18 units depending on configuration.

Input Peripherals

DualShock 4 Interface

The system natively supports DS4 controllers via Bluetooth.

  • L-Stick: Translation (X/Y)
  • R-Stick: Rotation (Yaw/Pitch)
  • R1/L1: Height Adjustment
  • Triangle: Toggle Gait Mode
